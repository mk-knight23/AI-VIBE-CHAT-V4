# Contributing to AI-VIBE

Thank you for your interest in contributing! 🎉

## Getting Started

1. **Fork** the repository
2. **Clone** your fork: `git clone https://github.com/YOUR_USERNAME/REPO.git`
3. **Create a branch**: `git checkout -b feat/your-feature`
4. **Make your changes** with meaningful commits
5. **Push** and open a **Pull Request**

## Development Setup

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Commit Convention

We follow [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add new AI provider
fix: resolve streaming timeout
docs: update README with examples
chore: upgrade dependencies
```

## Code Standards

- TypeScript strict mode
- ESLint + Prettier (run `npm run lint`)
- Add tests for new features
- Update README if adding features
